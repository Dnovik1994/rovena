# Onboarding — первые шаги пользователя

1. **Вход через Telegram**
   - Откройте Mini App и авторизуйтесь через Telegram WebApp.
   - Для локального запуска используйте `VITE_TG_INIT_DATA` в `.env`.
2. **Завершите онбординг**
   - Следуйте подсказкам мастера, чтобы подтвердить профиль.
3. **Добавьте прокси**
   - Admin → Proxies → Create → Validate.
4. **Добавьте аккаунт**
   - Accounts → Create → привязать прокси при необходимости.
5. **Запустите прогрев**
   - Accounts → Start warming.
6. **Создайте кампанию**
   - Campaigns → Create → указать source/target и лимиты.
7. **Запустите кампанию**
   - Campaigns → Start и следите за прогрессом.

Если возникают ошибки доступа — проверьте токен, статус пользователя и тарифные лимиты.
